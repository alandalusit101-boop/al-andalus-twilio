# AL Andalus - Twilio Integration Demo

This is a simple demo project for AL Andalus platform, showing how Twilio is used
inside the system to send verification codes (OTP) to users via SMS.

## Features
- Send SMS verification codes
- Simple Node.js backend structure
- Secure API usage

## How Twilio will be used in the real project
The AL Andalus platform uses Twilio for:
- Account verification
- Login 2FA
- Order notifications for customers

Only users who register inside the platform receive messages.

## Demo Code Files
The `/src/sendOtp.js` file contains the sample code used for sending OTPs.

This project is only intended to show Twilio the type of code we use
for account verification and SMS messages.
